# ASCII Paint - Консольный графический редактор

## Краткое описание проекта

ASCII Paint - это консольный редактор для создания ASCII-арта. Программа позволяет рисовать символом `#` на холсте размером 80×24 символов, перемещаясь стрелками и нажимая пробел.

## Возможности

- Рисование точек
- Рисование линий (алгоритм Брезенхема)
- Заливка области (flood fill)
- Два режима работы: рисование и стирание
- Сохранение и загрузка рисунков в текстовый файл
- Отмена последнего действия (Ctrl+Z)

## Применённые паттерны проектирования

- **Memento** - для сохранения состояния и отмены действий
- **State** - для управления режимами рисования
- **Command** - для обработки пользовательских команд

## Системные требования

- **Windows** 7/10/11 (программа использует Windows API)
- **PowerShell** или **CMD** (Git Bash НЕ поддерживается)
- Компилятор с поддержкой C++17 (MinGW, MSVC)
- CMake 3.10 или выше (для сборки через CMake)

## Сборка проекта

## Управление программой

| Клавиша | Действие |
|---------|----------|
| ↑ ↓ ← → | Движение курсора |
| ПРОБЕЛ | Рисование / стирание точки |
| R | Переключение режима (рисование/стирание) |
| L | Рисование линии (выбрать 2 точки пробелом) |
| F | Заливка области |
| C | Очистка холста |
| Ctrl+Z | Отмена последнего действия |
| S | Сохранить рисунок в файл `drawing.txt` |
| O | Загрузить рисунок из файла `drawing.txt` |
| Q | Выход из программы |

### Пример рисования прямоугольника:

1. Нажмите `L` для входа в режим линии
2. Переместите курсор в точку (10, 5) и нажмите `ПРОБЕЛ`
3. Переместите курсор в точку (30, 15) и нажмите `ПРОБЕЛ`
4. Будет нарисован прямоугольник

### Пример заливки:

1. Нарисуйте замкнутую фигуру
2. Переместите курсор внутрь фигуры
3. Нажмите `F` - область будет залита символом `#`

### Пример сохранения и загрузки:

1. Нарисуйте что-нибудь
2. Нажмите `S` - рисунок сохранится в файл `drawing.txt`
3. Нажмите `C` - холст очистится
4. Нажмите `O` - рисунок загрузится обратно

## Известные особенности

- Программа работает **только в Windows** (PowerShell или CMD)
- **Git Bash не поддерживается** из-за использования Windows API
- Файл сохранения `drawing.txt` создаётся в той же папке, откуда запущена программа
- История отмены хранит до 20 последних действий

## Устранение неполадок

| Проблема | Решение |
|----------|---------|
| `g++: command not found` | Установите MinGW или добавьте его в PATH |
| `cmake: command not found` | Установите CMake с официального сайта |
| Не видно курсор при рисовании линии | Убедитесь, что используете обновлённую версию `main.cpp` |
| Кракозябры в консоли | Используйте PowerShell или CMD вместо Git Bash |
